Mission

The Player Project creates Free Software that enables research in robot and sensor systems, including the Player robot server and the Stage multi-robot simulator.

Released under the GNU General Public License, all code from the Player Project is free to use, distribute and modify. Player Project software is developed by an international team of robotics researchers and used at labs around the world.
